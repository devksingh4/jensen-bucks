# Running Tests
1. Download the test input files from [Google Drive](https://drive.google.com/file/d/11zPDbnNsq_wnbC4lzJzC5HNNi2QkxLsC/view?usp=sharing) and place them in the data folder.
2. Compile the code, following the instructions from the top-level README.
3. Run `test.bat` on a compatible machine.
